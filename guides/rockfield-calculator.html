Rockfield Calculator

Multiplicative exclusive skill scaling. Example: if base were 30% and skill level is 3 (+6%), new base is 30% × 1.06 = 31.8%. Default base here is 27%, but you can change it below if needed.

Defaults to 27% (Rockfield description). Change if your tests show a different base.
Output is a single effective % of each enemy stack’s max HP.

Effective Damage

Formula

base = (Base Damage % / 100)
skillBonus = 0.02 × skillLvl
baseWithSkill = base × (1 + skillBonus)
afterLevel = baseWithSkill × (0.5)^(max(0, enemyLvl - yourLvl))
finalPct = afterLevel × min(1, yourMarch / enemyMarch)
